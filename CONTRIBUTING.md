# Contributing

XAIPT-GUARD Public Research Edition welcomes improvements that strengthen clarity, safety, and public understanding.

Accepted contribution areas:

- UI/UX polish
- documentation clarity
- governance-flow simulation
- accessibility improvements
- audit visualization
- policy-safe wording

Not accepted in this public repo:

- payment processing integrations
- exploit code
- bypass logic
- stealth surveillance behavior
- banking impersonation
- private XAIPT internals
