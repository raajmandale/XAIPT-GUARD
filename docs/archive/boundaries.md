# Boundaries

XAIPT-GUARD Public Research Edition is designed to be clear, responsible, and policy-safe.

## It does not

- process payments
- integrate with UPI
- replace banking apps
- bypass operating system security
- hide surveillance behavior
- provide anti-hacker internals
- expose private XAIPT architecture

## It does

- demonstrate governance flows
- explain authority-before-execution
- show audit visibility
- model delayed execution states
- present a public-safe UX pattern

## Why boundaries matter

The deeper XAIPT family contains private R&D directions. This repo exists to make the public concept understandable without weakening the private system strategy.
